Empirical CDF plot

ggplot(df, aes(x = INFg)) + 
  stat_ecdf() + 
  scale_y_continuous(n.breaks=10) + 
  ggtitle('ECDF') + 
  ylab('Density')

Inference: Empirical CDF
1. 14% of the data has values less than 0
2. Flat line on density=0.0 from -1.2 to -0.6 suggests that there are no ppints in this region; flat line after 4.5 suggests that there are very few data points with values less than -0.5 and greater than 4.5 i.e low density regions
3. Less than 1% values in the INFg column have values greater than 5
4. Because of high slope of the ECDF plot from -0.6 to 4, we can say that this is the region where most of the points lie, around 95% of the data(0.95-0.0)
5. There are no values less than -0.6 and more than 8.1
6. The ECDF plot is very smooth here which suggests that there are very high number of rows in dataframe

Density Plot

ggplot(df, aes(x = INFg)) +
  geom_density() +
  geom_rug(aes(y = 0), sides = "b", position = position_jitter(height = 0)) +
  scale_y_continuous(n.breaks=5) +
  ggtitle('Density Estimate') + 
  ylab('Frequency')

Inference: Density Estimate
1. The peak at 2.8 suggests that most of the datapoints have value 2.8
2. There are two significant peaks(-0.4, 2.8) in the Density plot so we can sat that the data is bimodal
3. From the rug plot, the region from -0.6 to 5.3 has most of the points as we can not see any individual whiskers in this region whereas after 5.3 the rug plot is not that dense hence there are comparatively less points here
4. The upper tail is close to 0 which means very few points lie towards the upper tail

Inference: QQ Plot
1. We can see that the lowest point is falling on the y=x line hene, CD4 and CD8 both has same lowest values
2. CD8 increases by a polynomial factor from the lowest point until CD4=0, after that CD8 increases by approximately a constant factor as CD4 increases(parallel to y=x)
